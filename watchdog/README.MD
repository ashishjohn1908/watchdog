# WatchDog
WatchDog is library that can be added in any project, so as to get basic device details in which the app is installed.

## Getting Started
1. [Download](https://developer.android.com/studio/index.html) the Android Studio. 
2. Clone repository.
	* i. HTTP :: http://git.finoit.com/WatchDogs/WatchDog-Android.git
	* ii. SSH :: git@git.finoit.com:WatchDogs/WatchDog-Android.git
3. Switch to <development> branch for latest code. 
4. Build the project in Android Studio. 
5. Always create feature branch of your task & then create merge request to develop branch.

## Branches in use
**master** - Base branch.
**develop** - Branch with latest code.
**staging** - Branch with last store version code.
**production** - Branch with last store version  code.
